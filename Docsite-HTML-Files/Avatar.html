Avatar

Avatars are images that users can set as their profile picture. They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.

Villainess Provides two kinds of Avatars , Round and Text respectively.

Round Image Avatar

There are images in four different sizes, to use them you need to include a common class round-img and use avatar-size-lg , avatar-size-md, avatar-size-sm, avatar-size-xs respectively as displayed below.

Text Avatar

There are images in four different sizes, to use them you need to include a common class round-img and use avatar-size-lg bg-color, avatar-size-md bg-color, avatar-size-sm bg-color, avatar-size-xs bg-color
